# Agent Skills Repository

This repository contains reusable AI agent skills, organized under the `skills/` directory.

## What's Inside

- `skills/convex-ddd-architecture`: A Convex architecture skill focused on Domain-Driven Design boundaries, repository patterns, adapters, triggers, and migration-safe evolution.

## Repository Structure

```text
skills/
  <skill-name>/
    SKILL.md
    supporting-docs.md
```

Each skill folder includes a `SKILL.md` entrypoint and optional supporting references.
